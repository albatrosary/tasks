## Install Library

### gulp

```bash
$ npm rm -g gulp
$ npm install -g gulp-cli
```

### typescript

```bash
$ npm install -g typings, typescript
$ typings init
$ tsc --init
```

### npm

```bash
$ npm i --save-dev lite-server
$ npm i --save-dev concurrently
$ npm i --save-dev typescript
$ npm i --save-dev typings
$ npm i --save-dev es6-shim
$ npm i --save-dev es6-promise
$ npm i --save-dev ts-node
$ npm i --save-dev gulp-typescript

```
